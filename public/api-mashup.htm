Feed API do Mashup para aplicações de terceiros

A api do Mashup possibilita que você integre o conteúdo do mashup em seu website ou aplicativo.

Localização do feed API

A maneira mais fácil de vocé acessar um feed ê através do icone localizado no rodapé da página do mashup

Parâmetros para consultar API

Para acesso ao feed de qualquer página do mashup basta adicionar na url o parâmetro format=rss. Ex:

  • mashup.cancaonova.com/?format=rss
  • mashup.cancaonova.com/PHN?format=rss

No segundo ex, PHN é uma tag que pode ser substituida por uma outra tag ou um conjunto de tags
ex: mashup.cancaonova.com/ {tag[+tag+...+tag]} ?format=rss

Formato da resposta

A resposta de toda consulta através da API será nos formatos MediaRSS. Futuramente outros formatos estarão disponíveis.

Futuras funcionalidades da API

  • Adicionar paginação na API
    ex: mashup.cancaonova.com/PAGE/2?format=rss
  • opção para retornar apenas video, foto e texto de conteúdo no feed
    ex: mashup.cancaonova.com/VIDEO/page/2?format=rss
  • resposta em outros formatos
