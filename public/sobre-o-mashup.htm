MASHUP.cancaonova.com

Mashup multimídia para a cobertura online dos eventos da Canção Nova

Objetivo

Reunir o conteúdo multimídia referente a Canção Nova existente nos sites externos.

Benefícios

Oferecer ao público um único endereço na internet para acompanhar o que acontece na Canção Nova, com acesso as diversas mídias fornecida pelos maiores sites da internet.

Conceito

Mashup é um website ou uma aplicação web que usa conteúdo de mais de uma fonte para criar um novo serviço.

fonte http://pt.wikipedia.org/wiki/Mashup

Tag é uma palavra-chave (relevante) ou termo associado com uma informação (ex: uma imagem, um artigo, um vídeo) que o descreve e permite uma classificação da informação baseada em palavras-chave.

fonte http://pt.wikipedia.org/wiki/Tag_(metadata)

Consumer-Generated Media (CGM) ou mídia gerada pelo consumidor é um termo utilizado para descrever o conteúdo que é criado e divulgado pelo próprio consumidor... Os consumidores utilizam todas as ferramentas disponíveis (Messenger, sites, blogs, e-mails, mensagens, celulares, etc.) para divulgar, sobretudo, suas experiências pessoais e opiniões em relação a produtos, serviços, marcas, empresas, notícias.

fonte wikipedia.org Consumer Generated Media

Recursos

  • Contribuição de conteúdo do usuário
  • Opção de multiplos layouts
  • Versão mobile do site
  • Embed do conteúdo do Mashup em blogs ou sites através da opção EMBED do CollIris
  • Interface media center através do CollIris
  • API do Mashup para outros desenvolvedores e/ou aplicativos fazerem uso do conteúdo

Diferencial de outras coberturas

  • A fonte de dados é exclusivamente de sites de terceiros (em outros domínios), como youtube.com, flickr.com, delicious.com, twitter.com
  • O conteúdo é composto de conteúdo curto, uma micro-media:
    • vídeo — YouTube
    • foto — Flickr
    • bookmark — Delicious
    • microblog — Twitter
  • Permitirá a navegação por tags, para organizar o conteúdo, um sistema semântico, obtido através da utilização de tags, Ex: mashup.cancaonova.com/pentecostes
  • Internacionalização, com o sistema de tags, o conteúdo será definido através do idioma utilizado na tag, ex: em português =padre+leo+sabado, em inglês = priest+leo+saturday
  • conteúdo gerado pelo usuário

Futuras funcionalidades

  • Adicionar outras fontes de conteúdo e mídia
  • Opção de uma interface Media Center, própria do mashup, para substituir o CoolIris
  • Opção de embed, própria do mashup, para substituir o CoolIris
  • Atualização AJAX do conteúdo da página, para poder acompanhar em tempo real o conteúdo sem precisar recarregar a página
  • melhorar a performace e o tempo de resposta da página
